# equip-1

- Part one of computer equipment series
- Records tapeless through digital out onto sd card

# parts

- Raspberry Pi CM4
- FireWire Card
- Custom PCB
- Screen + Buttons
- Printed case
- Ports USB-C and FireWire
- Battery
